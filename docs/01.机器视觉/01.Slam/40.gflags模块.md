## 安装

- 下载
  ```shell
  git clone https://github.com/gflags/gflags.git
  wget https://github.com/gflags/gflags/archive/master.zip
  ```

- 编译安装
  ```shell
  cd gflags
  mkdir build
  cd build
  cmake ..
  make
  sudo make install
  ```
