## 1 String 字符串

### 1.1 string 和 int 互换

```c
string s="123";
int a=atoi(s.c_str());

int i = 123;
std::string s = std::to_string(i);
```

### 1.2 string 是否包含某子字符串
```cpp
std::string a = "abcdefg";
std::string key = "bcd";
v = a.find(key);
if (v == std::string::npos) // 不存在
    std::cout << "not found" << std::endl;
else
    std::cout << "found" << std::endl;
```
