# 3 字体大小
字体大小有下面几种情况：

	\tiny
	\scriptsize
	\footnotesize
	\small
	\normalsize
	\large
	\Large
	\LARGE
	\huge
	\Huge

使用如下：
|`\scriptsize A=B`|$ \scriptsize A=B $|
|--|--|
|`\large A=B`|$ \large A=B $|
