# 关闭“系统出现异常”的提示框

#### ubuntu 使用一段时间后，总是出现“检测到系统错误”的提示框，其日志保存在 `/var/crash/` 中，此提示框可以禁用：

- 安装gksu
  ```bash
  sudo apt install gksu  #gksu是用来执行图形的（GUI）程序
  ```
- 修改 apport 文件
  ```bash
  $ gksu gedit /etc/default/apport
  #将enabled = 1 修改为下图的 0，然后保存，以后就不会推送系统程序出现问题了
  ```
  重启就不会出现提示框了
